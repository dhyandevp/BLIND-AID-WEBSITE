Introduction

Welcome to the Drishya Sahayak developer and user manual. This device is designed to be open and hackable for those who wish to extend its capabilities.

Initial Setup

1. Charge the device fully using the provided USB-C cable.

2. Put on the bone conduction headset.

3. Long press the power button on the compute unit for 3 seconds.

Note: The first boot may take up to 2 minutes as the AI models allow-map themselves to memory.

Firmware Updates

To update your device manually via CLI:

$ ssh blind-aid@192.168.1.X $ sudo blind-aid-update --stable > Checking for updates... > Downloading v1.0.3... [100%] > Installing... > Done. Please reboot.

Developer API

Access the raw object detection stream via WebSocket:

const ws = new WebSocket('ws://device-ip:8080/stream'); ws.onmessage = (event) => { const data = JSON.parse(event.data); console.log('Detected:', data.objects); };
